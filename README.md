# ElementInfo

Плагин **ElementInfo** предназначен для анализа и отображения информации об элементах в приложениях на платформе .NET Framework 4.8.
Работает в системе MagicEntry

## Возможности

- **Выбор элементов:**  
  Позволяет выбрать один или несколько элементов для анализа.  
  _Сообщение:_ `Выберите элементы (Esc - отмена)`

- **Анализ текста:**  
  Открывает диалог для ввода текста, содержащего ID элементов или имя вида.  
  _Сообщения:_  
  - `Введите текст для анализа:`  
  - `Неверный формат текста`  
  - `В тексте не найдено имя вида в формате 'Активный вид: [название]'`  
  - `В тексте не найдены ID элементов в формате 'ID: [число]'`

- **Копирование информации:**  
  Позволяет скопировать собранную информацию в буфер обмена.  
  _Сообщения:_  
  - `Скопировать текст в буфер обмена?`  
  - `Текст скопирован в буфер обмена`  
  - `В буфере обмена нет текста`

- **Отображение информации:**  
  Показывает подробную информацию об элементах, их местоположении, рабочем наборе, имени, ID, активном виде и пользователе.  
  _Форматы:_  
  - `Место расположения: {0}`  
  - `Рабочий набор: {0}`  
  - `Имя: {0}`  
  - `ID: {0}`  
  - `Активный вид: {0}`  
  - `Пользователь: {0}`

- **Обработка ошибок:**  
  Сообщает о проблемах при работе с элементами или текстом.  
  _Сообщения:_  
  - `Элементы с указанными ID не найдены`  
  - `Вид с указанным именем не найден`  
  - `Не выбрано ни одного элемента`

## Особенности

- Все сообщения и шаблоны форматирования хранятся централизованно в классе `Messages`.
- Поддерживаются сценарии: выбор элементов, анализ текста, копирование информации, обработка ошибок.
- Все действия сопровождаются информативными сообщениями для пользователя.

## Требования

- .NET Framework 4.8
